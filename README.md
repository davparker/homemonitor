homemonitor
===========

Author: Kamilo Amir

Version: alpha

Home Monitor App for Splunk

Welcome to Home Monitor for Splunk! This tool will allow you to visualize the traffic that is coming to your home modem (DSL, FiOS or other).  You only have to point the syslog information to your instance of Splunk and you'll be able to immediately be able to see the data flowing in.
